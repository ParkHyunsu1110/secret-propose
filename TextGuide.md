# 프로포즈 콘텐츠 가이드

이 파일을 수정한 뒤 `npm run generate:data` 실행 시 `src/main/resources/data.sql`이 자동 생성됩니다.

---

## 카드 뉴스

각 카드는 `### 번호`로 구분합니다. 번호 순서대로 정렬됩니다.
- **사진**: `- 사진:` 필드에 파일명(쉼표 구분) 또는 범위(`1-87` 형식) 지정. `1-87`은 memory-001.jpg ~ memory-087.jpg 자동 확장.
- 사진 파일은 `frontend/public/images/`에 넣어주세요.

### 1
- 사진: 1-87

## 편지

인사말, 맺음말, 본문을 작성합니다. 본문은 빈 줄로 문단을 구분합니다.

- 인사말: 사랑하는 당신에게
- 맺음말: 영원히 사랑하는, 당신의 사람으로부터
